# Piniglo

Piniglo la ung dung nhan dang trang phuc tu net ve chuot, dung du lieu QuickDraw va giao dien web HTML/CSS tone hong trang.

## Cong nghe

- Flask
- PyTorch
- HTML, CSS, JavaScript

## Nhom nhan hien tai

- `Áo thun`
- `Quần dài`
- `Áo khoác`
- `Quần short`
- `Áo len`
- `Giày`
- `Tất`
- `Đồ lót`
- `Mũ`
- `Ba lô`

## Chay ung dung

```bash
python app.py
```

Mo trinh duyet tai `http://127.0.0.1:5000`.

## Train lai model

```bash
python scripts/train_quickdraw.py
```

Script se tu tai du lieu QuickDraw dang `numpy_bitmap` ve thu muc `data/quickdraw/` va luu model vao `models/`.
